Welcome Sensei!

This is BA OST Index, a project created by sctop in an attempt to track relationships between each track and the stories, battles, as well as the backgrounds, characters and UIs!


Supported Data

Currently the website has nearly-full support for text-only data. The related images are expected to update afterwards.

  • Background
  • Character
  • Event
  • Main Mission/Story
  • Tag (not in use)
  • Tracks
  • UI
  • Videos

You can also navigate the website through the navigation section on the left.

BA OST Index is a project created by sctop.

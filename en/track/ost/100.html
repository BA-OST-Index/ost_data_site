Undefined Behaviour is the No. 100 track in the OST.

The 100th track in Blue Archive, used during the battle with Hod. "Undefined Behaviour" is a programming term, used to describe the result of executing a program whose behavior is prescribed to be unpredictable, in the language specification to which the computer code adheres.
